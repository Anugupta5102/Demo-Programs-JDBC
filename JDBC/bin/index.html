Welcome to the Hotel Management System

Our Hotel Management System simplifies the booking and management of hotel reservations.
With easy access to all essential functionalities, it ensures seamless customer service and efficient management.

Key Features

  • Reserve a Room
  • View Current Reservations
  • Room Number Retrieval
  • Update Reservation Details
  • Delete Reservations
  • Make Payments

Technical Details

  • Built using Java, JDBC and MySQL
  • Java Database Connectivity
  • Database Management
